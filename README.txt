This repository contains a web-based interactive dashboard for analyzing donation transactions using Streamlit. Built as part of a real-world internship project at Lazismu Banyumas, the dashboard processes donation data collected from the SobatBerbagi.com platform using both QRIS and manual transfer methods. It provides key insights into donor behavior, transaction trends, campaign performance, and donation method preferences for supporting data-driven fundraising strategies.
